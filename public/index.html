CloudNote D1

Create and share notes by URL slug. The browser encrypts/decrypts; the server stores only ciphertext in Cloudflare D1.

How it works

  • You choose a slug and a strong password.
  • Your browser derives a key (PBKDF2) and encrypts content (AES‑GCM).
  • The server stores only encrypted data (ciphertext, IV, salt).

Tip: treat the slug as public. The password is what protects your content.
